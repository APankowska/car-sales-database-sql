## SQL project for a car dealership (Glam Cars) involving relational database design, data population, and sales analysis queries.

# Project Overview
SQL project based on a car dealership scenario (Glam Cars). The aim is to design and implement a relational database and write SQL queries to analyse car sales data.

# Repository Contents
  - ['schema.sql'](schema.sql) - table definitions with primary and foreign keys
  - ['data.sql'](data.sql) - sample data used for testing queries
  - ['queries'](Queries.md) - SQL queries answering the assignment requirements

# Database Design
  - Tables: Centre, Salesperson, Customer, Carsale
  - Relationship implemented according to the given ERD

# Technologies
  - Oracle SQL

# Notes
This project was originally developed as part of an academic assessment.
